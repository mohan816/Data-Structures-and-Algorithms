Stack:
It is an Abstract Data Type, Which means we have to focus on the functions, not how internally this data structure is implemented.
It follows last in first out implementation method(LIFO)

Basic Operations:
push(): To insert an element in the stack
pop(): To remove the top most element in the stack
top(): To get the top most element in the stack
size(): To get the size of the stack, mean number of elements in the stack
isEmpty(): Returns true if the stack is empty else it returns false
